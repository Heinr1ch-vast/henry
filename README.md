# Business Services Hub

A simple static business services portal created in the `business-website` workspace.

## Files

- `index.html` — business services portal layout and content
- `styles.css` — responsive styling for desktop and mobile
- `script.js` — mobile navigation and contact form feedback

## Features

- Email account creation required to access portal information
- Clear, accessible business service categories
- Industry-specific guidance for startups, retail, and service businesses
- Contact form with business type entry for better onboarding
- Mobile-friendly navigation and layout

## Run locally

1. Open `index.html` in your browser.
2. Or use a simple local server, for example:
   - `python -m http.server` (Python 3)
   - `npx http-server .`
